Advancing sceince with Accelerated Machine Learning (ASAML)

Accelerated Machine Learning applied to fundamental science experiements

ASAML is a group funded by an NSF HDR focused on applying accelerated machine learning to two big data science experiements: The Large Hadron Collider (LHC) and The Laser Interferometer Gravitational-Wave Observatory . The goal for this work is to both apply machine learning and hardware based machine learning acceleration to enhance the speed and quality of detector reconstruction. This work is affiliated with the large Fast Machine Learning lab and the NSF CSSI.

Given the unique potential for scientific discovery with the Large Hadron Collider, and gravitational wave astrophysics, we aim to accelerate the capabilities of these experiments through the exploitation of deep learning algorithms combined with modern parallel processing technology. We will build upon our recent work developing high quality deep learning algorithms for real-time data analytics of time-series and image datasets. We will also build on our recent work using Field Programmable Gate Arrays (FPGAs) to accelerate low-latency inference of machine learning algorithms to minimize future computing demands. Our central goal is to reduce overall computing latency. This is quickly becoming a critical issue for particle physics and gravitational wave astrophysics. The open source tools to be developed as part of these activities will be readily shared with LIGO, LHC, and LSST to be used as core data analytics algorithms.

These activities respond to a series of NSF-funded workshops that this science team has organized to identify key areas that will benefit from the use of deep learning algorithms coupled with heterogeneous computing utilizing FPGAs, and aim to initiate a paradigm shift in computing and data analytics for large scale facilities across science domains. Ultimately, we aim to create an institute of hardware-based ML-acceleration that extends across science disciplines providing concrete solutions to exponentially growing data challenge needs across all domains of science.
